# 03 - 文件操作

<!-- STEP1-DONE: 2026-04-07 -->

## 核心概念

现代 Web 应用可以通过 File API 和 File System Access API 读写本地文件。

## 要点记录

- File API: 读取用户选择的文件
- File System Access API: 直接读写文件系统
- Blob 和 File 对象处理二进制数据
- 下载文件：创建临时链接触发下载
