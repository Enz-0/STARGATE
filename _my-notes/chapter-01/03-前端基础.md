## 一、HTML 骨架结构

### 1.1 什么是 HTML

HTML（HyperText Markup Language，超文本标记语言）是网页的**结构层**，负责定义页面的内容和语义结构。

### 1.2 标准 HTML 文档结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
</head>
<body>
    <!-- 页面可见内容放在这里 -->
    <h1>Hello World</h1>
    <p>这是我的第一个网页。</p>
</body>
</html>
```

### 1.3 各标签详解

| 标签 | 名称 | 作用 |
|:---|:---|:---|
| `<!DOCTYPE html>` | 文档类型声明 | 告诉浏览器这是 HTML5 文档，必须放在第一行 |
| `<html>` | HTML 根元素 | 所有内容的容器，`lang` 属性指定页面语言 |
| `<head>` | 文档头部 | 包含元数据（metadata），不显示在页面上 |
| `<body>` | 文档主体 | 页面上可见的所有内容 |
| `<meta charset="UTF-8">` | 字符编码 | 指定文档编码为 UTF-8，确保中文正常显示 |
| `<meta name="viewport">` | 视口设置 | 适配移动设备，确保响应式布局 |
| `<title>` | 页面标题 | 显示在浏览器标签页和搜索结果中 |

### 1.4 head 中常见的其他元素

```html
<head>
    <!-- 字符编码 -->
    <meta charset="UTF-8">
    
    <!-- 视口设置（移动端适配） -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- 页面描述（SEO 用） -->
    <meta name="description" content="这是一个示例网页">
    
    <!-- 页面标题 -->
    <title>我的网站</title>
    
    <!-- 引入外部 CSS -->
    <link rel="stylesheet" href="styles.css">
    
    <!-- 内部 CSS -->
    <style>
        body { background: #f0f0f0; }
    </style>
    
    <!-- 引入外部 JavaScript -->
    <script src="app.js" defer></script>
</head>
```

### 1.5 常用 HTML 标签速查

| 标签 | 作用 | 示例 |
|:---|:---|:---|
| `<h1>` ~ `<h6>` | 标题，h1 最大 | `<h1>主标题</h1>` |
| `<p>` | 段落 | `<p>一段文字内容</p>` |
| `<a>` | 超链接 | `<a href="https://example.com">点击访问</a>` |
| `<img>` | 图片 | `<img src="photo.jpg" alt="描述文字">` |
| `<div>` | 区块容器，用于布局 | `<div class="container">...</div>` |
| `<span>` | 行内容器，用于小范围样式 | `<span class="highlight">高亮</span>` |
| `<ul>` / `<ol>` | 无序列表 / 有序列表 | `<ul><li>项目1</li></ul>` |
| `<li>` | 列表项 | 配合 ul 或 ol 使用 |
| `<button>` | 按钮 | `<button>点击我</button>` |
| `<input>` | 输入框 | `<input type="text" placeholder="请输入">` |
| `<form>` | 表单 | `<form action="/submit">...</form>` |

## 二、CSS 引入方式

CSS（Cascading Style Sheets，层叠样式表）是网页的**表现层**，负责控制页面的视觉样式。

### 2.1 三种引入方式对比

| 方式 | 优先级 | 适用场景 | 推荐度 |
|:---|:---:|:---|:---:|
| **外部样式表** | 低 | 大型项目，多页面共享样式 | ⭐⭐⭐ 强烈推荐 |
| **内部样式表** | 中 | 单页面特殊样式，快速测试 | ⭐⭐ 偶尔使用 |
| **行内样式** | 高 | 动态样式（JS 控制），紧急修复 | ⭐ 不推荐 |

### 2.2 方式一：外部样式表（External）✅

将 CSS 代码写在独立的 `.css` 文件中，通过 `<link>` 标签引入。

**HTML 文件（index.html）**：
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的页面</title>
    <!-- 引入外部 CSS 文件 -->
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
```

**CSS 文件（styles.css）**：
```css
/* 页面背景 */
body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
}

/* 标题样式 */
h1 {
    color: blue;
    text-align: center;
}
```

**优点**：
- ✅ 样式与结构分离，代码更清晰
- ✅ 一个 CSS 文件可被多个 HTML 页面引用
- ✅ 便于维护和团队协作
- ✅ 浏览器会缓存 CSS 文件，提高加载速度

---

### 2.3 方式二：内部样式表（Internal）

将 CSS 代码写在 HTML 文件的 `<style>` 标签内。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的页面</title>
    <style>
        /* 内部 CSS */
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: blue;
            text-align: center;
        }
    </style>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
```

**适用场景**：
- 单页面网站
- 快速原型测试
- 页面特有的样式（不与其他页面共享）

---

### 2.4 方式三：行内样式（Inline）❌

直接在 HTML 标签的 `style` 属性中写 CSS。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的页面</title>
</head>
<body>
    <!-- 行内样式 -->
    <h1 style="color: blue; text-align: center;">Hello World</h1>
    <p style="color: gray; font-size: 14px;">这是一段文字</p>
</body>
</html>
```

**缺点**：
- ❌ 样式与结构混合，难以维护
- ❌ 优先级过高，难以覆盖
- ❌ 无法复用相同样式
- ❌ 增加 HTML 文件体积

**仅适用于**：
- JavaScript 动态修改样式
- 邮件模板（部分邮件客户端不支持外部 CSS）

## 三、第一个 "Hello World" 页面

### 3.1 项目结构

```
hello-world/
├── index.html      # 主页面文件
└── styles.css      # 样式文件
```

### 3.2 完整代码

**index.html**：
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>我的第一个网页</title>
    <!-- 引入外部 CSS -->
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <h1>🎉 Hello World!</h1>
        <p>欢迎来到 Web 开发的世界！</p>
        <p class="highlight">这是我用 HTML 和 CSS 创建的第一个页面。</p>
    </div>
</body>
</html>
```

**styles.css**：
```css
/* 页面基础样式 */
body {
    font-family: 'Microsoft YaHei', Arial, sans-serif;
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    min-height: 100vh;
    margin: 0;
    display: flex;
    justify-content: center;
    align-items: center;
}

/* 内容容器 */
.container {
    background: white;
    padding: 40px 60px;
    border-radius: 10px;
    box-shadow: 0 10px 40px rgba(0, 0, 0, 0.2);
    text-align: center;
}

/* 标题样式 */
h1 {
    color: #333;
    margin-bottom: 20px;
    font-size: 2.5em;
}

/* 普通段落 */
p {
    color: #666;
    font-size: 18px;
    line-height: 1.6;
}

/* 高亮段落 */
p.highlight {
    color: #667eea;
    font-weight: bold;
    margin-top: 20px;
}
```

### 3.3 运行方式

1. 创建文件夹 `hello-world`
2. 创建 `index.html` 和 `styles.css` 两个文件
3. 将上述代码复制到对应文件中
4. 在 VS Code 中右键 `index.html` → "Open with Live Server"
5. 浏览器会自动打开并显示页面

**预期效果**：
- 页面中央显示白色卡片
- 紫色渐变背景
- 标题带 emoji 图标
- 文字居中对齐
