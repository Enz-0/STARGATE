## 🚀 学习顺序建议

### 第一步：验证环境（5分钟）
```bash
cd demo
node 01-环境验证.js
```
- 确保所有环境配置正确
- 如有问题，返回 `../01-开发环境.md` 查看解决方案

### 第二步：Git 实践（20分钟）
1. 打开 `02-Git工作流程.md`
2. 按手册逐步执行每个命令
3. 在注释标记处记录实际操作结果
4. 最终完成：本地仓库 → GitHub 推送

### 第三步：网页实战（15分钟）
1. 进入 `03-第一个网页/` 目录
2. 用 VS Code 打开 `index.html`
3. 右键 → "Open with Live Server"
4. 按 `index.html` 中的 `TODO` 标记修改代码

## ⚠️ 注意事项

1. **环境验证失败？**
   - 检查是否正确安装了 Node.js
   - 确认 Git 已添加到系统 PATH
   - 查看 `../01-开发环境.md` 的安装步骤

2. **Git 推送失败？**
   - 确认已配置 GitHub 账号
   - 检查远程仓库地址是否正确
   - 可能需要配置 SSH 密钥或使用 HTTPS 登录

3. **网页显示异常？**
   - 确保 `index.html` 和 `styles.css` 在同一文件夹
   - 检查 Live Server 是否正确运行
   - 尝试刷新浏览器缓存（Ctrl+F5）
