## 📚 本节内容

### manifest.json 基础结构

```json
{
    "manifest_version": 3,
    "name": "我的扩展",
    "version": "1.0.0",
    "description": "扩展的描述信息",
    "permissions": [
        "storage",
        "activeTab",
        "tabs"
    ],
    "action": {
        "default_popup": "popup.html",
        "default_icon": {
            "16": "icons/icon16.png",
            "48": "icons/icon48.png",
            "128": "icons/icon128.png"
        }
    },
    "background": {
        "service_worker": "background.js"
    },
    "content_scripts": [
        {
            "matches": ["<all_urls>"],
            "js": ["content.js"],
            "css": ["content.css"]
        }
    ]
}
```

### 核心组件说明

| 组件 | 作用 | 文件 |
|------|------|------|
| Manifest | 扩展的配置文件 | manifest.json |
| Popup | 点击图标弹出的界面 | popup.html/js/css |
| Background | 后台服务脚本 | background.js |
| Content Script | 注入页面的脚本 | content.js |
| Options | 扩展设置页面 | options.html |

### Manifest V3 变化

```javascript
// V2: background page
"background": {
    "scripts": ["background.js"],
    "persistent": false
}

// V3: service worker
"background": {
    "service_worker": "background.js"
}
```

### 权限声明

```json
{
    "permissions": [
        "storage",        // 存储数据
        "tabs",           // 操作标签页
        "activeTab",      // 当前活动标签
        "scripting",      // 注入脚本
        "notifications"   // 通知
    ],
    "host_permissions": [
        "https://*.example.com/*"
    ]
}
```
