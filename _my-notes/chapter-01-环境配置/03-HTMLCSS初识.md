# 03 - HTML/CSS 初识

> 🎯 **学习目标**：理解 HTML 骨架结构，掌握 CSS 引入方式。

---

## 一、HTML 骨架结构

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
```

---

## 二、CSS 引入方式

### 1. 外部样式表（推荐）
```html
<link rel="stylesheet" href="styles.css">
```

### 2. 内部样式表
```html
<style>
    body { background: #f0f0f0; }
</style>
```

### 3. 行内样式
```html
<p style="color: red;">红色文字</p>
```

---

## 三、常用 HTML 标签

| 标签 | 作用 |
|:---|:---|
| `<h1>`~`<h6>` | 标题 |
| `<p>` | 段落 |
| `<a>` | 链接 |
| `<img>` | 图片 |
| `<div>` | 区块 |
