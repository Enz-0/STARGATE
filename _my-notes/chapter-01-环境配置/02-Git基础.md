# 02 - Git 基础概念

> 🎯 **学习目标**：理解 Git 核心概念，掌握版本控制的基本原理。

---

## 一、Repository（仓库）

**定义**：存储项目代码及其完整历史记录的文件夹。

**创建方式**：
```bash
git init
```

---

## 二、Commit（提交）

**定义**：保存项目某一时刻完整状态的"快照"。

```bash
git add .
git commit -m "提交说明"
```

---

## 三、Branch（分支）

**定义**：代码的独立开发线。

```bash
git branch feature-name
git switch feature-name
```

---

## 四、Staging Area（暂存区）

**定义**：准备下次提交的文件临时存放区域。

```bash
git add filename  # 添加到暂存区
git reset filename  # 从暂存区移除
```
