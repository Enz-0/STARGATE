# 01 - 开发环境配置

> 🎯 **学习目标**：搭建现代化的 Web 开发环境，安装并配置必要的开发工具。

---

## 一、Visual Studio Code 安装

### 1.1 什么是 VS Code

Visual Studio Code（简称 VS Code）是由微软开发的免费开源代码编辑器，是目前最受欢迎的 Web 开发编辑器。

### 1.2 核心功能

| 功能 | 说明 |
|:---|:---|
| **Syntax Highlighting** | 语法高亮，用不同颜色显示代码 |
| **Auto-completion** | 自动补全，智能提示代码 |
| **Debugging Tools** | 调试工具，设置断点、逐步执行 |
| **Extensions** | 扩展系统，通过插件增强功能 |
| **Built-in Terminal** | 内置终端，无需切换窗口 |
| **Git Integration** | Git 集成，可视化版本控制 |

### 1.3 必装扩展

- **Live Server**：本地开发服务器，保存自动刷新
- **Prettier**：代码格式化
- **GitHub Copilot**：AI 代码助手

---

## 二、Git 安装与配置

### 2.1 什么是 Git

Git 是一个**分布式版本控制系统**，用于跟踪代码的变更历史。

### 2.2 安装验证

```bash
git --version
```

### 2.3 初始配置

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

## 三、Node.js 安装

Node.js 让 JavaScript 可以在**浏览器之外运行**。

```bash
# 验证安装
node --version
npm --version
```
