# 01 - 键盘事件

<!-- STEP1-DONE: 2026-04-07 -->

## 📚 本节内容

### 键盘事件类型

| 事件 | 触发时机 | 特点 |
|------|----------|------|
| `keydown` | 按键按下时 | 持续按住会重复触发 |
| `keyup` | 按键释放时 | 每个按键只触发一次 |
| `keypress` | 按键按下时 | 已废弃，不建议使用 |

### 事件属性

```javascript
document.addEventListener('keydown', (event) => {
    event.key;       // 按键字符 (如 "a", "Enter", " ")
    event.code;      // 物理按键代码 (如 "KeyA", "Enter")
    event.keyCode;   // 数字代码 (已废弃)
    event.which;     // 同 keyCode (已废弃)
    
    event.ctrlKey;   // Ctrl 是否按下
    event.shiftKey;  // Shift 是否按下
    event.altKey;    // Alt 是否按下
});
```

### 常用按键判断

```javascript
document.addEventListener('keydown', (e) => {
    // 回车键
    if (e.key === 'Enter') {
        console.log('回车被按下');
    }
    
    // 空格键
    if (e.code === 'Space') {
        console.log('空格被按下');
    }
    
    // 方向键
    if (e.key === 'ArrowUp') {
        console.log('向上箭头');
    }
});
```

### 阻止默认行为

```javascript
document.addEventListener('keydown', (e) => {
    // 阻止 F5 刷新
    if (e.key === 'F5') {
        e.preventDefault();
    }
});
```
