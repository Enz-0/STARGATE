# 01 - 变量与类型

> 🎯 **学习目标**：掌握 JavaScript 变量声明和数据类型。

---

## 一、变量声明方式

### let（推荐）
```javascript
let name = 'Alice';
name = 'Bob'; // ✅ 可以重新赋值
```

### const（常量）
```javascript
const PI = 3.14159;
PI = 3.14; // ❌ 错误，不能重新赋值
```

### var（旧方式，避免使用）
```javascript
var count = 10; // 存在变量提升问题
```

---

## 二、数据类型

| 类型 | 示例 | 说明 |
|:---|:---|:---|
| **String** | `'Hello'` | 字符串 |
| **Number** | `42`, `3.14` | 数字 |
| **Boolean** | `true`, `false` | 布尔值 |
| **Undefined** | `undefined` | 未定义 |
| **Null** | `null` | 空值 |
| **Object** | `{ name: 'Tom' }` | 对象 |
| **Array** | `[1, 2, 3]` | 数组 |

---

## 三、类型检测

```javascript
typeof 'Hello';     // "string"
typeof 42;          // "number"
typeof true;        // "boolean"
typeof {};          // "object"
```
