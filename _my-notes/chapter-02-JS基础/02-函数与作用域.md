# 02 - 函数与作用域

> 🎯 **学习目标**：掌握 JavaScript 函数的各种写法。

---

## 一、函数声明方式

### 函数声明
```javascript
function greet(name) {
    return `Hello, ${name}!`;
}
```

### 函数表达式
```javascript
const greet = function(name) {
    return `Hello, ${name}!`;
};
```

### 箭头函数（推荐）
```javascript
const greet = (name) => `Hello, ${name}!`;
```

---

## 二、箭头函数特性

```javascript
// 单个参数可省略括号
const square = x => x * x;

// 多行需要大括号和 return
const sum = (a, b) => {
    const result = a + b;
    return result;
};
```

---

## 三、闭包概念

```javascript
function counter() {
    let count = 0;
    return function() {
        return ++count;
    };
}

const myCounter = counter();
myCounter(); // 1
myCounter(); // 2
```
