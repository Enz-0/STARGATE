# 03 - 控制流

> 🎯 **学习目标**：掌握条件判断和循环语句。

---

## 一、条件判断

### if...else
```javascript
const age = 18;

if (age >= 18) {
    console.log('成年人');
} else if (age >= 13) {
    console.log('青少年');
} else {
    console.log('儿童');
}
```

### 三元运算符
```javascript
const status = age >= 18 ? '成年' : '未成年';
```

---

## 二、循环语句

### for 循环
```javascript
for (let i = 0; i < 5; i++) {
    console.log(i);
}
```

### while 循环
```javascript
let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}
```

---

## 三、数组方法

```javascript
const numbers = [1, 2, 3, 4, 5];

// forEach - 遍历
numbers.forEach(num => console.log(num));

// map - 映射
const doubled = numbers.map(num => num * 2);

// filter - 过滤
const evens = numbers.filter(num => num % 2 === 0);
```
